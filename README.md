

# Cobrapad
A beginner-friendly pygame wrapper for easy 2D game development.

## Installation
pip install cobrapad

## Usage
import cobrapad

screenX = 800
screenY = 600

cobrapad.open_screen()
cobrapad.screen.paint("dark_blue")
cobrapad.run()

## Note
- This library is still in beta
- Pydroid3 (Android) users may need to add import pygame at the top and run from the Pygame tab
